Broker Portal Guidance

Use our Broker Portal to get instant updates on cases 24/7

Broker Portal log in

ONLY FOR USE BY MORTGAGE INTERMEDIARIES

On this page
Broker Portal overview
FAQs

Once logged into the Broker Portal, you can do the following

Case updates

For new business and additional borrowing applications:

  • If we have requested further information from you, the Portal will confirm exactly what we need. In your ‘Submitted Cases’ you will find a ‘Case Tracking’ option. Select this and then click on the ’Upload document/further information’ tab. For additional borrowing you will need to log into the "Existing Customer Portal" to upload any documentation requested. 
  • Case tracking also shows you milestones which are updated in real time. They show a traffic light system so you can clearly see which stages of the application process are outstanding and what has been completed.
  • In the same ‘Case Tracking’ screen you can check the status of a valuation appointment and find out the returned valuation figure.
  • The ‘Case Tracking’ screen also show’s you the property address and product the application is proceeding on. This is useful if you have sent a request to change either and want to know if these changes have been processed yet.
  • Can’t see any updates? Check our processing timescales so you know when to expect to see an update on the portal. 

 

For Product Transfers, you can check progress in two places:​

  • Existing Customers Portal – you’ll see the product you applied for and when it’s due to start. Once the switch completes, the new product will show as your customer’s active rate.​

  • Case Tracking (in Submitted Cases) – this will show ‘Completion’ once the Product Transfer has gone live. It won’t show which product you applied for.​

These systems don’t confirm when your customer has signed the paperwork.​
However, you’ll be copied into the email confirming the paperwork has been signed.

Making changes to the application

  • For new business applications, you can easily change the product applied for in the ‘Case Actions’ screen.
  • For additional borrowing applications you can request changes to products by uploading a memo to your client's document upload portal under the other documents section.

For guidance on how to change the rate on a product transfer application, please see our Product Transfer Hub.

Access and maintenance

  • You can give administrator access to your colleagues so they can use the Portal to get updates on your applications. Your administrator simply needs to fill out their details on the Administrator registration form. You will then need to provide them with access to your applications using the ‘Case Access Management’ option.
  • You can update your details in the ‘Maintain Details’ tab on our portal. If you are changing your last name, email address or firm address, any applications submitted prior to this change will no longer be visible on the portal.
  • Please note: Administrator  access cannot be granted for the "Existing Customer Portal" at this time therefore any additional borrowing application tasks including document upload will need to be completed via a broker access login. 

FAQs

AIPs

What should I do if an AIP refers?

If an AIP refers, you will need to submit a full application, send in the required case packaging and then a decision will be given by an underwriter within our current standard processing timescales.

Can I get an AIP certificate

Where an AIP accepts, you can download an AIP certificate. You can't get a copy once the application has gone to full submission (FMA). If you need to get an AIP for an existing case, you’ll need to generate a new AIP.

How long is an AIP valid for?

Our AIPs are soft footprints which will be converted to a hard footprint at full mortgage application.

  • Residential AIPs:

The AIP is guaranteed for 30 days (including the day it was created), provided no changes are necessary. Progressing to full mortgage application within this period will mean no further credit search is completed.

  • Buy to Let AIPs:

Due to our dynamic stress rates within our BTL mortgages the AIP is valid while that product (or an alternate lower rate) is available, up to a maximum of 30 days. If a new higher rate is taken to application or rates increase, this will reduce the overall maximum lend available.

Common Portal queries

Trying to apply and unable to get past the Direct Debit mandate section?

Please select “Direct Debit Information” this will generate a pop up with the Direct Debit mandate. Please read and close this pop up and this will allow you to proceed. If you don’t get a pop up, please also check that pop ups are not blocked on your browser.

Why has my case disappeared after I have submitted a full mortgage application?

Once a case has been submitted it will move from ‘recent’ cases to ‘submitted cases’ which can take up to 24hrs. If the application is a porting case this can take 72 hours to show as this will need to be keyed manually.

Sending us packaging

What file formats can I upload documents in?

Answer: JPG, PNG and PDF.

What is the maximum file size for an upload?

Answer: 8mb.

Can I view documents I have uploaded to a case?

No, you will only be able to see confirmation that the documents have been uploaded.

Administrator access

I cannot find my administrator on case access management?

Check your administrator has registered and ask them to locate you under Case Access Management first

Mortgage documentation

Can I get a copy of the application once a mortgage has completed?

Once an application has completed you won’t be able to download a copy for the full mortgage application. Make sure you download a copy of this before the mortgage completes as we can’t send this to you after completion.

FAQ - Uploading PDFs

What should I do if the PDF I’m trying to upload is over the maximum file size limit?

Here are two ways you can split a PDF:

1. Split a PDF Using Microsoft Edge (built‑in on Windows)

Steps:

•Open the PDF in Microsoft Edge (Right‑click the file → Open with → Microsoft Edge)

•Press Ctrl + P.

•Choose Microsoft Print to PDF as the printer.

•Under Pages, choose a page range (e.g., 1–20).

•Click Print, then save the new PDF.

•Repeat for the next chunks (e.g., 21–40).

2. Split a PDF Using Google Chrome (built‑in on Windows)

Steps:

•Open the PDF in Google Chrome (Right‑click the file → Open with → Google Chrome)

•Press Ctrl + P.

•Change Destination to Save as PDF.

•Under Pages, select Custom. Type the page numbers you want in this new file (e.g. 1-20)

•Click Save, then save the new PDF.

Repeat for the next chunks (e.g., 21–40).

3. Use Adobe Acrobat (if you have it)

If you have Adobe Acrobat Standard/Pro, it has a dedicated split feature.

Steps:

•Open the PDF in Adobe Acrobat (Right‑click the file → Open with → Adobe Acrobat)

•Go to Tools → Organise Pages.

•Select Split.

•Choose how to split:

•by number of pages

•by maximum file size

•by top-level bookmarks

•Click Split.
